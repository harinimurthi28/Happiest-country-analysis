# 😊 Happiest Country Analysis  

## 📌 About the Project  
This project analyzes global happiness data to understand what factors make a country happier. It uses data visualization to present insights clearly and effectively.  

## 🎯 Objective  
- Analyze happiness scores of countries  
- Identify key factors influencing happiness  
- Compare countries and regions  
- Visualize data for better understanding  

## 🛠️ Technologies Used  
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## 📊 Dataset  
The dataset includes:  
- Happiness Score  
- GDP per capita  
- Social Support  
- Life Expectancy  
- Freedom  
- Generosity  
- Corruption  

## 📈 Analysis  
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization (bar charts, graphs)  
- Correlation analysis  

## 🔍 Key Insights  
- Countries with higher GDP and social support are generally happier  
- Life expectancy plays an important role  
- Developed countries rank higher in happiness index  

  

## 📌 Conclusion  
This project shows that happiness is influenced by economic, social, and health factors. Data visualization helps in understanding these insights easily.
